##远程桌面批量管理器（Remote Desktop Manager）
* 是基于C# 开发的一款工具 （免费，开源 ，可以根据自己需要增加功能）
* 软件的目的：让管理服务器更加方便，解放运维
* 支持同时多个主机链接
* 支持ping
* 支持中英双语切换
* 支持分组
* 支持主机颜色标记
